## Beskrivelse av spillet

Du er en ny ambisiøs student på IFI
Målet ditt er å fullføre utdanningen din og få en bachelor i informatikk
Før du kan få bachelorgraden, må du gjennom en rekke ulike utfordringer (emner)
Vær forsiktig! Gjør du for mange feil, risikerer du arbeidsløshet!
Samle pant for å fylle på med energi (HP), og få deg utdanning og jobb!
